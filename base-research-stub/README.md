# Base Research — компактный старт

Простой репозиторий для быстрого старта исследования блокчейна **Base**.

## Как использовать
1. Установите зависимости:
   ```bash
   python -m pip install -r requirements.txt
   ```

2. Установите переменную окружения `BASE_RPC_URL`:
   ```bash
   export BASE_RPC_URL="https://base-mainnet.rpc-url.example"
   ```

3. Запустите пример:
   ```bash
   python src/main.py
   ```
